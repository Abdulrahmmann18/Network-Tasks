# CCNA Projects

## Overview
This repository contains a collection of networking projects implemented for CCNA certification. Each project demonstrates practical skills in various networking protocols and configurations.

## Projects

### 1. Telnet
- **Description:** Configured secure Telnet sessions for remote device management.

### 2. Static Routing
- **Description:** Implemented static routes to ensure reliable network connectivity.

### 3. OSPF
- **Description:** Configured OSPF for dynamic routing and efficient path selection.

### 4. NAT
- **Description:** Set up Network Address Translation to enable internal network access to the internet.

### 5. VLAN
- **Description:** Created and configured VLANs to segment network traffic and enhance security.

### 6. DHCP
- **Description:** Configured DHCP server for automatic IP address allocation.
